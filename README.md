WordPress plugin creating searchable directories of mental health therapists, counselors, psychiatrists, and wellness coaches. Features filterable listings by specialty (anxiety, depression, trauma, couples, adolescent), insurance accepted, therapy modality (CBT, DBT, EMDR, somatic), telehealth availability, sliding scale fee acceptance, and languages spoken. Each profile includes credentials, license verification status, intake form, appointment booking integration with SimplePractice and TherapyNotes, and client-controlled review system. Supports HIPAA-compliant contact forms with secure messaging. Admin includes crisis resource widget, directory analytics, and therapist verification workflow. Schema markup for local SEO, PHP 8.1+, WordPress 6.x+.

== Installation ==

1. Upload the plugin folder to `/wp-content/plugins/`
2. Activate via WordPress admin > Plugins
3. Configure via Settings > Mental Health Therapy Directory Plugin
